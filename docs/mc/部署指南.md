# MC Token 部署指南

## 合约信息

- **总量**: 1.2 亿 MC
- **买入税**: 0%
- **卖出税**: 10% (2% 钱包1 + 3% 黑洞 + 3% 钱包2 + 2% 钱包3)
- **黑洞销毁阈值**: 1200 万 MC

## 部署命令

### 1. 编译

```bash
forge build
```

### 2. 部署

```bash
forge script script/mc/Deploy.s.sol:DeployScript \
  --rpc-url https://bsc-dataseed.binance.org \
  --broadcast
```

### 3. 部署后设置

```bash
# 模拟运行
forge script script/mc/Setup.s.sol:SetupScript \
  --rpc-url https://bsc-dataseed.binance.org

# 执行设置
forge script script/mc/Setup.s.sol:SetupScript \
  --rpc-url https://bsc-dataseed.binance.org \
  --broadcast
```

### 4. 合约验证

```bash
# 验证 MC 合约
forge verify-contract 0xb531613381ccE69DACdfe3693570f8cbf8BDA81f \
  src/mc/MC.sol:MC \
  --constructor-args $(cast abi-encode "constructor(address,address,address,address)" "0x8B1b198626F7b96Eb0C7C536bf68f4cB7095bC47" "0x8B1b198626F7b96Eb0C7C536bf68f4cB7095bC47" "0x8B1b198626F7b96Eb0C7C536bf68f4cB7095bC47" "0x10ED43C718714eb63d5aA57B78B54704E256024E") \
  --chain-id 56 \
  --watch

# 验证 TaxDistributor 合约
forge verify-contract 0x0c7d5a9A13828d0F6cEc06FBb60c1351842afb62 \
  src/mc/TaxDistributor.sol:TaxDistributor \
  --constructor-args $(cast abi-encode "constructor(address,address,address,address,address)" "0xb531613381ccE69DACdfe3693570f8cbf8BDA81f" "0x10ED43C718714eb63d5aA57B78B54704E256024E" "0xEA37DEb2F85aeda26372efb8F59EDd0F6aC74B19" "0x3dFc8bc2BD7d387638477dd6453c07EA61cF2622" "0xBf93Be345758CF027577705F981C6bE02fA88184") \
  --chain-id 56 \
  --watch
```
