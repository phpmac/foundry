# 快速指南

介绍环境的安装和基本使用

- [安装 foundry](https://getfoundry.sh/)
- `uv pip install slither-analyzer` 安装 slither-analyzer

## 脚本部署

使用 `--rpc-url` 参数指定网络, 支持两种方式:

1. **使用别名**: `forge script script/mc/DeployWithdraw.s.sol --rpc-url bsc_mainnet --broadcast`
2. **直接 URL**: `forge script script/mc/DeployWithdraw.s.sol --rpc-url https://bsc-dataseed.binance.org --broadcast`

当前配置的别名 (在 `foundry.toml` 中定义):
- `bsc_mainnet` - BSC 主网

## 快捷命令

```bash
# 最常使用
forge fmt && forge build && slither .

# 构建指定合约
forge build --build-info src/bank/Bank.sol

# 生成 JSON 报告
slither . --json slither-report.json

# 生成人类可读摘要
slither . --print human-summary

# 筛选特定风险 (如重入)
slither . --detect reentrancy-eth,reentrancy-no-eth,unchecked-lowlevel

# 查看合约信息/继承图
slither . --print contract-summary
slither . --print inheritance-graph
```
